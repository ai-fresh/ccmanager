FAQ

Is it safe to auto-accept Execute (Bash)?

The Execute tier gives Claude full terminal access. Only enable it if you trust the codebase and Claude's judgment. For most workflows, Read + Write is a good balance.

Does CC Gate work with multiple projects?

Yes. CC Gate monitors all sessions in ~/.claude/projects/ and displays them in a single list with per-project status.

What happens if CC Gate is closed?

Claude Code falls back to its default behavior — asking in the terminal. CC Gate is purely additive; closing it changes nothing.

Can I use it without auto-accept?

Absolutely. Keep all tiers disabled and use CC Gate purely as a session monitor and question management UI.

What data does CC Gate collect?

CC Gate reads local session data from ~/.claude/projects/. No data is sent to external servers. Everything stays on your Mac.
