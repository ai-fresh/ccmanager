GitHub Integration

Push commits, fork repositories, clone projects, and navigate to commits on GitHub.com — all without leaving CC Gate.

Overview

CC Gate integrates with GitHub to enable repository operations directly from the app. Once you configure a Personal Access Token, you can push commits after the AI Commit flow, fork repositories under your account, clone new projects, and jump directly to any commit on GitHub.com with one click.

Setting Up a GitHub Token

CC Gate uses a GitHub Personal Access Token (classic) for REST API operations. The token is stored in macOS Keychain — never in any file on disk or sent to any server other than api.github.com.

  1. Open GitHub token settings

    Go to github.com/settings/tokens, or navigate: GitHub → Settings → Developer settings → Personal access tokens → Tokens (classic).

  2. Generate a new token

    Click "Generate new token (classic)". Set a descriptive name like "CC Gate" so you remember what it's for.

  3. Set expiration

    Choose 90 days for better security, or "No expiration" if you prefer not to rotate it. A calendar reminder to refresh is recommended for time-limited tokens.

  4. Select required scopes

    Check repo (full repository access, including private repos). Optionally add delete_repo if you want CC Gate to be able to clean up forks. No other scopes are needed.

  5. Copy the token

    GitHub shows the token only once. Copy it immediately — it starts with ghp_. If you close the page without copying, you must generate a new one.

  6. Add token to CC Gate

    Open CC Gate → Settings → GitHub tab → paste the token into the "Personal Access Token" field → click Save token. The status indicator should turn green showing "Connected".

Security note The token is stored using macOS Keychain Services — the same mechanism used by Safari to store passwords. CC Gate never writes it to any file, and never transmits it to any server other than api.github.com during REST API calls.

GitHub Operations

Push Changes

After completing a commit in CommitSheet, a "Push" button appears showing how many commits are ahead of origin/main. Click it to push directly. If there are uncommitted changes, CC Gate shows a warning with an option to commit first ("Go to Commit") before pushing.

Fork Repository

Right-click any project in the CC Gate menu → Fork. CC Gate calls the GitHub API to fork the repository under your account. The remote URL is automatically extracted from the project's git configuration, so you don't need to type anything. The fork appears in your GitHub account within seconds.

Clone Project

Click the "+" button in the menu bar → Clone from GitHub → enter an owner/repo identifier or a full GitHub URL. CC Gate clones the repository into your configured default projects directory and immediately shows it in the project list.

Go to Commit on GitHub

After a successful push, a "View on GitHub" button appears. Clicking it opens the specific commit URL in your default browser, so you can share it, review the diff, or copy the commit SHA.

Privacy and Security

  • CC Gate does not read your GitHub username or password
  • SSH key authentication is handled entirely by macOS — CC Gate has no access to your SSH keys
  • For HTTPS remotes, login credentials are stored in macOS Keychain by git itself, not by CC Gate
  • The Personal Access Token is used exclusively for GitHub REST API calls (fork, clone list, repository info)
  • CC Gate does not send any data to third-party analytics or tracking services

Troubleshooting

"Not connected / API token not set"

Add your Personal Access Token in Settings → GitHub. The status badge will turn green once a valid token is saved.

Push rejected

This typically means the remote has commits your local branch does not have. Run git pull origin main (or the relevant branch) in your terminal, resolve any merge conflicts, then try pushing again from CC Gate.

Fork already exists

GitHub's API returns an error if a fork of the repository already exists under your account. Navigate to github.com/your-username/repo-name directly. CC Gate cannot merge or overwrite an existing fork.

Token expired or invalid

Generate a new token at github.com/settings/tokens following the setup steps above, then update it in Settings → GitHub → paste new token → Save token.
