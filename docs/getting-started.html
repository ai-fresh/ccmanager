Getting Started

CC Gate is up and running in about two minutes — here's everything you need to do.

1. Download & Install

1

Download CC Gate

Head to the Releases page on GitHub and download the latest .dmg file.

2

Drag to Applications

Open the downloaded file. A window appears — drag the CC Gate icon into the Applications folder shown next to it.

3

Launch CC Gate

Open your Applications folder and double-click CC Gate. If macOS says it can't be opened, right-click the icon and choose Open instead — then click Open again in the dialog.

Good to know: CC Gate lives in your menu bar — the row of small icons at the very top-right of your screen. It has no Dock icon, so don't worry if you don't see it there.

2. Connect to Claude Code

For CC Gate to work with Claude Code, you connect them once — it takes about 10 seconds.

  1. Click the CC Gate icon in the menu bar to open the popup.
  2. Click the gear icon to open Settings.
  3. Go to the Hook tab.
  4. Click Install hook.
  5. You'll see a green "Installed" badge appear — you're connected.
Important: Open a new Claude Code session after installing — the connection only works for sessions started after this step. If Claude Code is already running, just start a fresh session.

3. Allow accessibility access (optional)

Only needed if you want CC Gate to type answers for you directly in the terminal. Most people skip this step — you can always come back to it later.

  1. Open Settings in CC Gate.
  2. Go to the Permissions tab.
  3. Click Grant Accessibility Access and follow the macOS prompt.

4. Turn on Auto-Accept

Without Auto-Accept, Claude Code will stop and ask for permission every time it wants to read a file, write code, or run a command. Turning on a few levels makes the whole experience much smoother.

  1. Click the bell icon in the top-right corner of the CC Gate popup.
  2. Toggle Read on — this lets Claude read and search files without interrupting you. It's completely safe.
  3. Optionally toggle Write on too, so Claude can edit files without asking each time.
Tip: Start with just Read and IDE enabled. You can always add more levels later once you're comfortable with how CC Gate works.

You're all set!
