Settings Reference

Complete reference for every option in CC Gate's settings window, organized by tab.

Open Settings by clicking the CC Gate menu bar icon → the gear icon, or by pressing the keyboard shortcut shown in the menu. Settings are divided into tabs: General, Permissions, Hook, AI, GitHub, Pro, and About.

General Tab

⚙

General

Startup behavior, project defaults, editor preferences, and monitoring frequency.

Launch at Login Start CC Gate automatically when you log into macOS. CC Gate runs as a menu bar app with no Dock icon, so it stays out of the way. Default: Off
New Project Folder The default directory where CC Gate places new projects created via the Clone or New Project flow. You can always override per-project. Tap "Browse" to select a folder. Default: ~/Documents
Default Application Which editor or terminal to open when you click "Open" on a project. Options: VS Code, Cursor, Warp, iTerm2, Terminal. The selected app must be installed — CC Gate checks at launch and shows a warning if the selected app is not found. Default: VS Code
Monitoring — Refresh Interval How often CC Gate scans ~/.claude/projects/ for project changes, session status updates, and active Claude Code processes. Set with a slider from 5 to 60 seconds. Lower values make the status display more responsive; higher values reduce CPU and battery usage. Default: 15 seconds
Battery tip At the default 15s interval, CC Gate has minimal energy impact. If you are on battery or do not need real-time status, try 30s or 60s. At 5s, CC Gate runs frequent ps and file system calls which noticeably increase CPU usage.

Permissions Tab

🔒

Permissions

macOS system permission required for sending keyboard input to terminal windows.

Accessibility Access CC Gate needs Accessibility permission to send keyboard input to terminal windows when you answer a Claude Code question manually (for questions that are not auto-accepted). Without this, the "Answer" button in the Questions window will not be able to type your response into the terminal.
Grant button Opens System Settings → Privacy & Security → Accessibility. Find CC Gate in the list and enable the toggle. macOS may require your administrator password.
Restart App After granting Accessibility permission, click "Restart App" to reload CC Gate. The permission takes effect only after restart.
Auto-Accept users If you use CC Gate exclusively in auto-accept mode and never manually answer questions from the CC Gate UI, Accessibility permission is not required.

Hook Tab

🔗

Hook

Install and manage the permission hook that connects Claude Code to CC Gate.

Status badge Shows the current hook state. Green "Installed" means the hook is present and the version matches. Yellow "Outdated" means a newer hook version is available and the hook should be reinstalled. Red "Not installed" means the hook file or registration is missing.
Install / Uninstall button Install writes ~/.claude/hooks/ccmanager-permissions.sh and registers it in ~/.claude/settings.json under hooks.PermissionRequest. Uninstall removes both the file and the registration entry. Always use this button — never edit the hook file or settings.json directly.
Auto-repair If Claude Code resets or overwrites settings.json and removes the hook registration, CC Gate detects this on the next refresh cycle and automatically re-adds the registration entry. The hook file itself is not affected.

AI Tab

✨

AI

Configure the Anthropic API key used for AI Commit Message generation.

Enable AI commit messages Master toggle for the AI Commit Messages feature. Requires a Pro plan. When disabled, CommitSheet shows a manual commit message field only. Default: Off (requires Pro)
API Key Your Anthropic API key from console.anthropic.com. Stored in macOS Keychain, never written to disk. Starts with sk-ant-.
Test button Sends a minimal test request to the Anthropic API to verify the key is valid and has sufficient credits. Shows a green checkmark on success or an error message on failure.
Model Which Claude model to use for generating commit messages. Default: Claude Haiku for diff summarization chunks (fast and cheap), Claude Sonnet for the final combined message (higher quality). Both are configurable. Default: Haiku for chunks, Sonnet for final
Privacy CC Gate sends only the git diff and session context to the Anthropic API. No data is stored on Anthropic servers beyond the time required to process the request. Your code is not used for model training.

GitHub Tab

🐙

GitHub

Personal Access Token for GitHub REST API operations: push, fork, clone, and view commits.

Personal Access Token A GitHub Personal Access Token (classic) with repo scope. Required for push, fork, clone, and GitHub Release creation. Stored in macOS Keychain. Generate one at github.com/settings/tokens.
Save token / Clear Save adds the token to Keychain and shows a connected status. Clear removes the token from Keychain. CC Gate will show "Not connected" after clearing until a new token is saved.
Status Shows "Connected" with a green indicator when a valid token is saved, or "Not connected" with a gray indicator when no token is present.

Pro Tab

⭐

Pro

Manage your CC Gate Pro subscription and configure Pro features.

Plan status Shows your current plan: Free or Pro. Includes subscription renewal date for Pro subscribers.
AI Commit Messages Pro feature card. When on Pro plan, shows a "Configure" button that links to the AI tab for API key entry. When on Free plan, shows an upgrade prompt.
More Pro features coming soon Additional Pro-exclusive features are in development. Current Pro users will receive them automatically as updates are released.

About / Info Tab

ℹ

About

Version info, build details, links, and the Developer Mode easter egg.

Version number Current CC Gate version (e.g. "CC Gate 2.11.12"). Tap this label 7 times rapidly to unlock Developer Mode, which exposes the Test Runner and other experimental features. A "Developer Mode" checkmark appears after the 7th tap.
Build number Internal build identifier used when reporting bugs. Always include this in bug reports.
GitHub link Opens the public CC Gate repository and landing page at github.com/ai-fresh/ccgate.
Release notes Opens the GitHub Releases page where you can read the full changelog for each version.
Support / Report issue Opens the GitHub Issues page to file a bug report or feature request.
